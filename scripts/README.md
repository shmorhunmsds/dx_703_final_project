# Scripts

Helper scripts and experimental code for the project.

## Structure

- **`experiments/`** - Experimental model implementations and training scripts
  - `problem3_*.py` - Various iterations of Problem 3 CNN implementations
  - `custom_architectures.py` - Custom model architectures
  - `verify_pipeline.py` - Pipeline verification scripts

- **`utils/`** - Utility scripts and tools
  - `textcnn_improved.py` - Improved TextCNN architecture implementations
  - `vocab_check.py` - Vocabulary analysis utilities
  - `vocab_experiment.py` - Vocabulary size experiments
  - `cell*_corrected.py` - Corrected notebook cells
  - `test_environment.py` - Environment testing utilities
  - Installation scripts for PyTorch/TensorFlow

## Usage

Most scripts are designed to be imported or run standalone. See individual file docstrings for details.
