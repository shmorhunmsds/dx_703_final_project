# Documentation

Project documentation, notes, and analysis reports.

## Structure

- **`notes/`** - Development notes and reference guides
  - `CELL_CORRECTIONS.md` - Corrections and fixes for notebook cells
  - `QUICK_REFERENCE.md` - Quick reference guide
  - `COLAB_GIT_GUIDE.md` - Guide for using Git with Google Colab
  - `transformer_tokenizer_tfkeras_notes.md` - Notes on tokenizer integration

- **`analysis/`** - Analysis and diagnostic reports
  - `diagnose_model_issue.md` - Model diagnostic reports
  - `performance_bottleneck_analysis.md` - Performance analysis
  - `dataset_verification_checks.md` - Dataset verification results

## Purpose

These documents track development decisions, troubleshooting steps, and analysis findings throughout the project.
