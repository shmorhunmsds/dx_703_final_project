# Figures

Visualizations and plots generated during analysis and model training.

## Structure

- **`milestone1/`** - Exploratory data analysis visualizations
  - `category_comparison.png` - Category distribution comparison
  - `class_distribution.png` - Class distribution bar chart
  - `class_imbalance.png` - Class imbalance visualization
  - `missing_data.png` - Missing data patterns
  - `text_length_boxplots.png` - Text length distribution boxplots
  - `text_length_distributions.png` - Text length histograms

- **`milestone2/`** - Model training and evaluation plots
  - (Training curves, confusion matrices, etc.)

## Purpose

These figures support the analysis in the milestone notebooks and provide visual insights into data characteristics and model behavior.
